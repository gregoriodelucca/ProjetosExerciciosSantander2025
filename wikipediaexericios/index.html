Wikipedia — Enciclopédia Livre

Wikipedia é uma enciclopédia online colaborativa, construída e mantida por voluntários em todo o mundo. Ela é conhecida por seu vasto conteúdo, constantemente atualizado e disponível em muitos idiomas.

Este layout moderno busca oferecer uma experiência limpa e fácil de usar, mantendo a simplicidade e funcionalidade que tornam a Wikipedia uma das maiores fontes de conhecimento do planeta.

Este projeto é apenas uma demonstração simples com HTML e CSS para um portfólio ou projeto pessoal.
